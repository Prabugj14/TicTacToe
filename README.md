# Tic-Tac-Toe Game 🎮

A classic Tic-Tac-Toe game built with Python using the `tkinter` library for the graphical user interface. This project features a clean UI, score tracking, and automatic game reset functionality.

## Features ✨

- **Graphical User Interface**: Built with `tkinter` for a native desktop experience.
- **Scoreboard**: Tracks wins for Player X and Player O across multiple rounds.
- **Visual Feedback**: 
  - Winning lines are highlighted in **Green**.
  - Ties are highlighted in **Red**.
- **Auto-Restart**: The board automatically clears after a win or tie (with a 2-second delay).
- **Random Start**: The starting player is randomized at the beginning of each new game.

## Prerequisites 📋

- **Python 3.x** installed on your machine.
- **Tkinter**: Usually included with standard Python installations.
  - *Linux users*: If you encounter errors, install it via: `sudo apt-get install python3-tk`
